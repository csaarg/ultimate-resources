## 内容简介

　　网红级的开发手册，始于阿里内部规约，在全球 Java 开发者共同努力下，已成为业界普遍遵循的开发规范，涵盖编程规约、异常日志、单元测试、安全规约、MySQL 数据库、工程规约、设计规约七大维度。修炼一年终出关，这个版本将会给开发者带来更多惊喜。
